\doxysubsection{Detailed Description}
Represents an edge in the \mbox{\hyperlink{classDCEL}{DCEL}} data structure as a pair of half edges. 

An edge is defined as the line segment between two vertices in the polygon. It is represented by two \mbox{\hyperlink{classHalfEdge}{Half\+Edge}} objects in the \mbox{\hyperlink{classDCEL}{DCEL}} data structure, with one \mbox{\hyperlink{classHalfEdge}{Half\+Edge}} representing each directed end of the edge. The pair of \mbox{\hyperlink{classHalfEdge}{Half\+Edge}} objects are stored as a pair in the edge member variable of the \mbox{\hyperlink{classEdge}{Edge}} class.
