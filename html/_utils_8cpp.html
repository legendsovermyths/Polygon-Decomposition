Detailed Description

Implementation of various utility functions used in the DCEL data structure.

Includes functions for calculating orientation, finding all notches of a face, checking for intersection between two lines, and determining if a point is inside a polygon.

◆ getAllNotches()

vector< Vertex * > getAllNotches ( Face *  f)

Returns a vector of pointers to the Vertex objects that are considered notches within a given Face object.

Here, a notch is defined as a Vertex object in the Face object where two HalfEdge objects of the Face object meet at a reflex angle.

Parameters
fPointer to the Face object for which the notches need to be identified.
Returns
A vector of pointers to the Vertex objects representing the notches in the given face.
