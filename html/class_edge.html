Detailed Description

Represents an edge in the DCEL data structure as a pair of half edges.

An edge is defined as the line segment between two vertices in the polygon. It is represented by two HalfEdge objects in the DCEL data structure, with one HalfEdge representing each directed end of the edge. The pair of HalfEdge objects are stored as a pair in the edge member variable of the Edge class.
