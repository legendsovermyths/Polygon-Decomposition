\doxysubsection{Detailed Description}
Implementation of various utility functions used in the \mbox{\hyperlink{class_d_c_e_l}{DCEL}} data structure. 

Includes functions for calculating orientation, finding all notches of a face, checking for intersection between two lines, and determining if a point is inside a polygon.
