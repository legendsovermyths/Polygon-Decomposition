\doxysubsection{Detailed Description}
Represents an edge in the \mbox{\hyperlink{class_d_c_e_l}{DCEL}} data structure as a pair of half edges. 

An edge is defined as the line segment between two vertices in the polygon. It is represented by two \mbox{\hyperlink{class_half_edge}{Half\+Edge}} objects in the \mbox{\hyperlink{class_d_c_e_l}{DCEL}} data structure, with one \mbox{\hyperlink{class_half_edge}{Half\+Edge}} representing each directed end of the edge. The pair of \mbox{\hyperlink{class_half_edge}{Half\+Edge}} objects are stored as a pair in the edge member variable of the \mbox{\hyperlink{class_edge}{Edge}} class.
