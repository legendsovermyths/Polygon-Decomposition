\doxysubsection{Detailed Description}
Represents a directed edge in a polygon. 

A \mbox{\hyperlink{class_half_edge}{Half\+Edge}} stores information about one half of an edge in the polygon. Each edge in the polygon is represented by two \mbox{\hyperlink{class_half_edge}{Half\+Edge}} objects, one for each direction along the edge. The \mbox{\hyperlink{class_half_edge}{Half\+Edge}} class stores information about the vertex on which the edge is incident, the face on the right side of the edge, and pointers to the other half of the edge and its neighboring edges.
