\doxysubsection{Detailed Description}
Algorithm to decompose a simple polygon into a number of convex polygons. 

The \mbox{\hyperlink{class_m_p1}{MP1}} class constructor takes a doubly connected edge list (\mbox{\hyperlink{class_d_c_e_l}{DCEL}}) as input and initializes the notches vector by calling the \mbox{\hyperlink{_utils_8cpp_af0e208152de81a50ea2d788ccf79180b}{get\+All\+Notches()}} function on the external face of the input polygon. It then initializes the polygon data member and prints out the indices of all the notches in the input polygon.

The \mbox{\hyperlink{class_m_p1_a91a97580147e9c43f3b697ba6ae23edb}{prime\+MP1()}} function implements the main \mbox{\hyperlink{class_m_p1}{MP1}} algorithm. It first initializes an empty list L, and then adds the first two vertices of the polygon to L. It then iterates over the remaining vertices of the polygon and adds them to L if they form a \char`\"{}convex corner\char`\"{} with the previous two vertices. If the polygon has only one edge, it returns true. If L has two or fewer vertices, it returns false. Otherwise, it repeatedly removes \char`\"{}inessential vertices\char`\"{} from L until L has only three vertices left. An inessential vertex is a vertex that does not affect the monotonicity of the polygon. This is done by creating a rectangle that bounds the current chain of vertices in L, and then checking if any notches are inside the rectangle but outside the chain of vertices. If such a notch is found, the vertex of L that is farthest from the segment connecting the notch to the opposite vertex of the rectangle is removed from L. Finally, if L has more than three vertices, an edge is added from the first to the last vertex of L, and the function returns false.

The \mbox{\hyperlink{class_m_p1_a0959662a52206f1a9b4509e274274354}{merge()}} function implements the merging algorithm to remove inessential diagonals. It iterates over all the edges that ~\newline
 were added to the polygon in the previous step and removes them if they are not \char`\"{}essential\char`\"{} diagonals. An essential diagonal is a diagonal that connects two vertices that form a convex corner with each other.
