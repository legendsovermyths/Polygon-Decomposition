\doxysubsection{Detailed Description}
This class is used to make a \mbox{\hyperlink{class_rectangle}{Rectangle}} whose the edges are parallel to the coordinate axis. 

A class for representing a rectangle whose edges are parallel to the coordinate axes.

The class stores the minimum and maximum coordinates along the x and y axes of the polygon. The coordinates are specified using the parameters x1, y1, x2, and y2. The class also has a method is\+Inside() that takes a \mbox{\hyperlink{class_vertex}{Vertex}} pointer and returns true if the vertex lies inside the rectangle, and false otherwise.
